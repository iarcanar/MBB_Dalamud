Code changes in MBB.py show_splash()

หลัง load + resize splash, ก่อน draw version text:

# Load + resize meteor logo (height-based; width preserves aspect ratio)
# IMPORTANT: use mbb_meteor.png (the MBB brand logo) — NOT mbb_icon.png
# (that's the Dalamud plugin icon, a generic square placeholder).
from PIL import Image
icon_path = resource_path("assets/mbb_meteor.png")
icon = Image.open(icon_path).convert("RGBA")

# Pick HEIGHT first; let width follow source aspect (meteor logo is ~3:2)
icon_h = max(30, int(font_size * 1.2))   # 36px at font 30 (Option B)
icon_w = int(icon_h * (icon.width / icon.height))
icon = icon.resize((icon_w, icon_h), Image.Resampling.LANCZOS)

# Layout: [icon] [gap] [version] — centered as a group
gap = 14
group_w = icon_w + gap + text_w
group_left = (new_width - group_w) // 2

# Icon vertically centered with text (use icon_h, not icon_size)
icon_x = group_left
icon_y = visible_top + (text_h - icon_h) // 2

# Adjust text position (no longer just centered — group is centered instead)
text_x_new = group_left + icon_w + gap - bbox[0]

# Paste icon onto rounded splash (alpha-aware)
rounded.alpha_composite(icon, (icon_x, icon_y))

# Then draw version text at text_x_new (replace existing text_x)
# Glow + sharp text logic unchanged
📋 ขั้นตอนต่อไป: ลองคลิก Screen + Icon-size combos ดู positioning ใน canvas → ตัดสินใจ option → บอกผมว่า "ใช้ Option B" (หรือ A/C) → ผมจะแก้ MBB.py:show_splash() ตาม spec + bump version + rebuild + test

TUI · ตำแหน่งการแสดงผลเริ่มต้น

🧪 Proto · #2
Goal: แสดงตำแหน่ง+ขนาดเริ่มต้น (default) ของ TUI ทั้ง 4 โหมด เมื่อยังไม่มี tui_positions / tui_geometries บันทึกไว้ — สูตรดึงจาก translated_ui.py (Dialog/Choice, Tkinter) และ pyqt_ui/dissolve_overlay.py (Battle/Cutscene, PyQt6) โดยตรง. คลิกโหมดเพื่อสลับดู — หรือ "Show all" ดูพร้อมกันทั้ง 4
💡 ลากกรอบ TUI ในแคนวาสได้ — พิกัด (x, y) มุมขวาบนของแต่ละกรอบเป็นพิกัดจริงตามขนาดจอที่เลือก. ลากเสร็จแล้วบอกตำแหน่งที่ต้องการได้เลย → ผมจะเอาไปใช้เป็น default ใน show_for_mode() / dialog initial geometry

Why two renderers? Dialog/Choice ใช้ Tkinter เพื่อ multi-line + scrollbar + outline shadow. Battle/Cutscene ใช้ PyQt6 DissolveOverlay เพื่อ gradient alpha ที่ขอบซ้าย-ขวา (Tkinter transparentcolor เป็น 1-bit ไม่รองรับ gradient). ChatType dispatcher อยู่ใน translated_ui.update_text decorator
