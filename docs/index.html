1 ติดตั้ง Dalamud

มี Dalamud อยู่แล้ว? — ถ้าคุณใช้ Plugin อื่นบน Dalamud อยู่ก่อนแล้ว ข้ามขั้นนี้ไป Step 2 ได้เลย

Dalamud คือ Plugin Framework สำหรับ FFXIV ที่ชุมชนผู้เล่นทั่วโลกใช้งานกันอย่างแพร่หลาย ช่วยให้ติดตั้ง Plugin และ Mod ต่างๆ ได้สะดวก รวมถึง MBB Dalamud นี้

คำให้สัมภาษณ์ของ Naoki Yoshida เรื่อง third-party tool

"การใช้ Third-party tool ที่ไม่ได้ส่งผลกระทบหรือสร้างความเดือดร้อนให้ผู้เล่นคนอื่น เป็นสิ่งที่ทำได้ และจะไม่มีการดำเนินการใดๆ ต่อผู้ใช้งาน"

— Naoki Yoshida, ผู้กำกับและโปรดิวเซอร์ Final Fantasy XIV

3 เพิ่ม Plugin DLL ใน Dalamud

เพิ่มไฟล์ DLL ของ Plugin เข้า Dalamud เพื่อให้เกมโหลด MBB Bridge ตอนเริ่มเกม

  1. 1 เปิดเกม → พิมพ์ /xlsettings ใน chat
  2. 2 ไปที่แท็บ Experimental
  3. 3 ที่หัวข้อ Dev Plugin Locations → กด + เพิ่มแถวใหม่ → กด ... Browse
  4. 4 ไปที่ folder MBB ที่แตกไว้ → เข้าไปใน folder plugin\ → เลือกไฟล์ DalamudMBBBridge.dll
  5. 5 กด Save and Close → พิมพ์ /xlplugins → Plugin จะปรากฏในรายการ "Dev Tools"

ตัวอย่าง path (เปลี่ยนตามที่คุณแตกไฟล์):

D:\MBB\plugin\DalamudMBBBridge.dll
4 ตั้งค่า MBB.exe ให้พร้อมใช้งาน

สรุปวิธีง่ายๆ

เปิดเกม › คลิกขวา MBB.exe › Run as Admin เล่นได้เลย!

สำหรับผู้ที่ต้องการรันโปรแกรมผ่าน shortcut ในเกม ได้ อ่านเพิ่มเติม »

ขั้นนี้ทำ 2 อย่าง — (1) บอก Dalamud ว่า MBB.exe อยู่ตรงไหน เพื่อให้กดเปิดจากในเกมได้ (ใส่ใน macro hotbar slot สะดวก) และ (2) ตั้งให้รันด้วยสิทธิ์ Administrator เพื่อให้ฟีเจอร์ keyboard hook (เช่น ซ่อน TUI ตอนกด WASD วิ่ง) ทำงานได้

ไม่ตั้ง path ก็ใช้งานได้ — ดับเบิ้ลคลิก MBB.exe ในโฟลเดอร์เปิดเองหลังเปิดเกมก็ได้ แต่ "Run as Admin" แนะนำให้ตั้งเสมอ ไม่งั้นปุ่มลัด WASD ใช้ไม่ได้

4A ตั้ง Run as Administrator (ครั้งเดียวจบ)

  1. 1 คลิกขวา ที่ MBB.exe ในโฟลเดอร์ → Properties (คุณสมบัติ)
  2. 2 แท็บ Compatibility → ติ๊ก ☑ "Run this program as an administrator" → Apply → OK

ตั้งครั้งเดียวพอ — ครั้งต่อๆ ไปดับเบิ้ลคลิก MBB.exe จะถาม UAC ทีนึงแล้วเปิดเป็น admin ให้เลย

ทำไมต้องตั้งเป็น Admin?

Keyboard hook ของ MBB (เช่นการตรวจจับปุ่ม WASD เพื่อซ่อน TUI ขณะวิ่ง) ต้องใช้สิทธิ์ admin เพื่อ "เห็น" คีย์บอร์ดได้ตอนที่ FFXIV เป็น window ที่ active — เพราะเกมรันด้วยสิทธิ์ admin ของตัวเอง โปรแกรมข้างนอกที่จะ hook ได้ต้องอยู่ระดับเดียวกันหรือสูงกว่า

4B ตั้ง Path สำหรับเปิดจากในเกม (ทางเลือก · กดดูรายละเอียด)

ตั้งครั้งเดียว เพื่อเปิด MBB จากในเกมได้ — ใช้คำสั่ง /mbb launch หรือใส่เป็น macro ใน hotbar slot สะดวก

  1. 1 ในเกม → พิมพ์ /mbb ใน chat → หน้าต่าง config เปิดขึ้น
  2. 2 หาส่วน 📁 MBB Path Configuration → กด Browse... → เลือกไฟล์ MBB.exe → กด Save Path

ตัวอย่าง path (เปลี่ยนตามที่คุณแตกไฟล์):

D:\MBB\MBB.exe

หลังตั้ง path เสร็จ — ใช้คำสั่ง /mbb launch หรือกดปุ่ม "Launch MBB Application" ในหน้า /mbb เปิด MBB ได้จากในเกมเลย (ใส่ใน macro hotbar ก็ได้)

5 ใส่ API Key แล้วเล่นเลย!

เปิด MBB App → ใส่ Gemini API Key → กด START → เล่นเกมได้ทันที

รับ Gemini API Key ฟรี

ไปที่ Google AI Studio (aistudio.google.com) → สร้าง API Key → วางใน Settings ของ MBB

For existing users · อัพเดตในคลิกเดียว

อัพเดตเป็นเวอร์ชั่นใหม่

ในโฟลเดอร์ MBB ของคุณจะมีไฟล์ MBB-Updater.exe ติดมาด้วย ใช้สำหรับอัพเดตเป็นเวอร์ชั่นใหม่ โดยไม่ต้องดาวน์โหลด zip ใหม่ทั้งหมด

  1. 1 ปิด FFXIV และ MBB.exe ถ้าเปิดอยู่
  2. 2 เข้าโฟลเดอร์ MBB → ดับเบิลคลิก ที่ MBB-Updater.exe
  3. 3 หน้าต่าง Updater แสดงเวอร์ชั่นปัจจุบัน + เวอร์ชั่นล่าสุด → กด "Update Now" → รอดาวน์โหลดเสร็จ
  4. 4 เสร็จแล้วกด "🚀 Launch MBB" เพื่อใช้งานเวอร์ชั่นใหม่ทันที

ข้อมูลของคุณปลอดภัย — Updater เก็บ npc.json, npc_images/, settings.json, และ API key ไว้ครบ ไม่หาย
