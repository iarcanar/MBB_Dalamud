## 🎯 Installation URL for FFXIV


### How to Add in FFXIV:

1. Open Dalamud settings: `/xlsettings`
2. Go to **"Experimental"** tab
3. Add the URL above to **"Custom Plugin Repositories"**
4. Save and close
5. Open plugin installer: `/xlplugins`
6. Search for **"MBB Dalamud Bridge"**
7. Click **"Install"**
