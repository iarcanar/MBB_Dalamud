Hard-Won Rules

⚡ Must-read สำหรับ AI agent ที่จะ dev ต่อ — กฎที่ถูกพิสูจน์มาแล้วจากบัคจริง.

Dalamud C# Bridge — Native Safety (v1.8.18)

AtkValue type-gate ไม่ใช่กฎเหมารวม — Talk addon คือข้อยกเว้น

AtkValue เป็น union → เช็ค .Type == AtkValueType.String ก่อนอ่าน .String.Value เฉพาะ addon ที่ report เป็น String จริง. ⚠️ Talk addon (dialogue, ChatType 61) ส่ง pointer ที่อ่านได้ที่ AtkValues[0]/[1] แต่ .Type ไม่ใช่ String → ใส่ gate เข้มจะ early-return ทิ้ง dialogue ทุกบรรทัด (regression v1.8.17 → revert v1.8.18). OnTalkAddonPreReceive ต้อง null-check pointer อย่างเดียว ไม่มี type gate. TalkSubtitle (cutscene 71) ตรงข้าม — [0].Type เป็น String จริง → คง gate ไว้. อย่า "harden" native path ที่ทำงานดีอยู่แล้วโดยไม่เทสต์ในเกม.

จับเฉพาะ type ที่ใช้ + Dispose ต้อง symmetric
  • Capture เฉพาะ dialogue (61) · battle (68) · cutscene (71) · choice (70); filter อย่างอื่นทั้งหมด — hook เกมออนไลน์ยิงข้อความท่วม, จับหมด = หายนะ. ห้ามเพิ่ม hook กว้างหรือ per-frame (PostUpdate/PreDraw) โดยไม่ขออนุมัติ.
  • ทุก addon listener ที่ register ต้องมี Dispose() unregister คู่กัน — TalkSubtitle เคย leak (v1.8.18 fix) → reload ปลั๊กอินแล้ว cutscene ถัดไปยิง handler บน instance ที่ตายแล้ว → crash.

Frozen-Mode (exe) Robustness (v1.8.18)

บัคที่เห็นเฉพาะตอนรันเป็น exe (มองไม่เห็นใน dev เพราะ CWD = python-app/ พอดี)
  • Atomic npc.json write — npc_data_manager.save() เขียน .tmp → fsync → os.replace() (atomic). กัน crash กลางเขียนทำไฟล์เสีย (อาการ "npc.json ไม่อ่าน" / NPC Manager แครช).
  • Loader .get() hardening — npc.json ไม่ครบ → degrade เป็น empty แทน KeyError.
  • CWD normalization — os.chdir(get_app_dir()) ต้น __main__ → settings/fonts/logs เซฟถูกที่ทุก launch path (แก้อาการ "setting ไม่เซฟผ่าน exe").
  • Stub schema fix — ensure_npc_file_exists() สร้าง stub ตรง schema จริง (กัน KeyError ตอน install ไม่ครบ).

Tkinter Threading

Main thread ONLY
  • winfo_exists(), attributes(), update(), destroy() — main thread ONLY
  • Thread-based animation → ใช้ root.after() recursive chain แทน
  • Tkinter + PyQt6 hybrid shutdown บางครั้ง crash ด้วย PyEval_RestoreThread GIL error — รู้แล้วว่าไม่อันตราย

PyQt6 Gotchas

Gotcha #1

QSS overrides setFont

Panel-level QSS subtree cascade override setFont() silently.

Fix: apply font-family/font-size ผ่าน QSS, หรือ pre-render เป็น QPixmap ผ่าน QPainter (bypass QSS pipeline)

Gotcha #2

QGraphicsDropShadowEffect rasterizes children

All descendants rasterize together — children-of-shadow ติด bounding rect → ghost outlines

Fix: sibling overlap pattern (Polaroid). Action buttons เป็น siblings ของ shadowed card, ไม่ใช่ children.

Gotcha #3

Hover Enter/Leave flickers

Overlapping siblings ทำให้ Enter/Leave loop ไม่จบ — cursor crossing onto sibling = Leave on parent → hide → cursor on parent → Enter → ...

Fix: timer-based geometry polling (60-80ms) แทน Enter/Leave events

Gotcha #4

QTimer.singleShot(0) from worker

Cross-thread QTimer.singleShot(0, ...) silently no-ops — fires on calling thread, ไม่ใช่ UI thread.

Fix: ใช้ pyqtSignal (auto-queued connection) สำหรับ cross-thread results

Gotcha #5

QtFontManager is lazy

Runs ตอน Settings/Font panel เปิดเท่านั้น. Components ที่ใช้ custom font ก่อน panel เปิด → font ไม่ load

Fix: Components call QFontDatabase.addApplicationFont() เอง (idempotent — Polaroid pattern)

Gotcha #6

App-level eventFilter from BG threads

App-level eventFilter รับ events จาก background threads (เช่น global keyboard hook). Exception propagate ไป Qt's C++ side → app terminate ทันที

Fix: wrap callback bodies ใน try/except + log

Win32 + Tkinter Don'ts

Don't 1 — WM_NCLBUTTONDOWN modal resize from Tk callback

PyEval_RestoreThread GIL fatal. SendMessageW blocks main thread inside modal loop; Tk window proc fires WM_PAINT/WM_SIZE back to NULL Python thread state → process terminates.

Don't 2 — SetWindowRgn during drag

CreateRoundRectRgn + redraw + update_idletasks() = jank. Re-apply only at stop_resize().

Don't 3 — transparentcolor for alpha gradient

1-bit color-key — partial-alpha pixels render as opaque squares. ใช้ PyQt6 frameless + translucent แทนสำหรับ alpha gradient.

Frozen-Mode npc.json Rule

Any code touching npc.json MUST use get_npc_file_path()

resource_path() resolves to immutable bundled snapshot — wrong for npc.json (user writes). Reserve resource_path() สำหรับ static assets (fonts, icons).

Verify: grep -rn 'resource_path.*npc' python-app/ ต้อง return zero matches

PyQt6 Version Pin

PINNED at 6.9.0 — DO NOT UPGRADE Qt 6.10+ frameless+translucent rendering regression — ทำให้ DissolveOverlay + Polaroid + main window กลายเป็นพื้นทึบสีดำ.
