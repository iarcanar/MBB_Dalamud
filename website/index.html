ฐานข้อมูลตัวละคร FFXIV ครบถ้วน — ชื่อ, เพศ, สรรพนาม, บุคลิก ทำให้ AI แปลได้ตรงคาแรกเตอร์ เหมือนซับไตเติลมืออาชีพ

Wide-Context Translation

Glass Mode

วิธีติดตั้ง

1

ติดตั้ง Dalamud

Dalamud คือ Plugin Framework สำหรับ FFXIV ที่ชุมชนผู้เล่นทั่วโลกใช้งานกันอย่างแพร่หลาย ช่วยให้ติดตั้ง Plugin และ Mod ต่างๆ ได้สะดวก รวมถึง MBB Dalamud นี้

"การใช้ Third-party tool ที่ไม่ได้ส่งผลกระทบหรือสร้างความเดือดร้อนให้ผู้เล่นคนอื่น เป็นสิ่งที่ทำได้ และจะไม่มีการดำเนินการใดๆ ต่อผู้ใช้งาน"

— Naoki Yoshida, ผู้กำกับและโปรดิวเซอร์ Final Fantasy XIV

ดาวน์โหลด XIVLauncher — Dalamud จะติดตั้งอัตโนมัติพร้อมกัน

ดาวน์โหลด XIVLauncher
2

ดาวน์โหลดและติดตั้ง MBB

ดาวน์โหลด MBB Installer (.exe) แล้วรันได้เลย ตัวติดตั้งจะจัดการทุกอย่างให้อัตโนมัติ — ทั้ง Python App และ Dalamud Plugin

ติดตั้ง MBB Python App (standalone, ไม่ต้องติดตั้ง Python)
ติดตั้ง Dalamud Plugin (MBB Bridge) โดยอัตโนมัติ
ตั้งค่า Path เชื่อมต่อให้เรียบร้อย

กำลังพัฒนา — Installer จะพร้อมให้ดาวน์โหลดเร็วๆ นี้

MBB Installer
3

ตั้งค่า Path ใน Dalamud Plugin

เปิด Dalamud Plugin ของ MBB แล้วบอก path ที่ติดตั้ง MBB.exe ไว้ เพื่อให้ Plugin รู้ว่าจะเชื่อมต่อไปที่ไหน

  1. 1 เปิดเกม → กด /xlplugins หรือ Dalamud Plugin List
  2. 2 ค้นหา MBB Dalamud → คลิก ⚙ Settings
  3. 3 ช่อง MBB App Path → วาง path ไปยัง MBB.exe
  4. 4 กด Save → Plugin พร้อมเชื่อมต่อ
C:\Program Files\MBB\MBB.exe

ใส่ API Key แล้วเล่นเลย!

Magicite Babel - Dalamud Edition ทำงานผ่าน Dalamud Framework

โปรแกรม Magicite Babel - Dalamud Edition ทำงานผ่าน Dalamud Plugin Framework ซึ่งเป็น third-party tool ที่ชุมชน FFXIV ใช้กันอย่างแพร่หลาย กระบวนการทำงานของ Plugin คือ:

  1. Dalamud Plugin ดึงข้อความจากเกมผ่าน Text Hook (ไม่แก้ไขไฟล์เกม)
  2. ส่งข้อความไปยัง Python App ที่ทำงานแยกต่างหาก
  3. Python App ส่งข้อความไปแปลด้วย Gemini AI
  4. แสดงผลคำแปลในหน้าต่าง overlay แยกต่างหาก

โปรแกรมนี้ ไม่ได้ให้ความได้เปรียบในการเล่นเกม ไม่ดัดแปลง UI ของเกม และไม่ส่งข้อมูลใดๆ กลับไปยังเซิร์ฟเวอร์ของเกม — เป็นเพียงเครื่องมือช่วยอ่านและทำความเข้าใจเนื้อเรื่อง

โปรแกรมนี้พัฒนาขึ้นด้วยความรักและชื่นชมในเกม Final Fantasy XIV เพื่อช่วยให้ผู้เล่นชาวไทยได้เข้าถึงเนื้อเรื่องอันลึกซึ้ง และสนับสนุนคอมมูนิตี้ FFXIV ในประเทศไทยให้เติบโตยิ่งขึ้น

คำถามที่พบบ่อย

Q: ต้องจ่ายเงินไหม?

ไม่ ทั้ง Plugin และ App ฟรี — เสียค่า Gemini API เฉพาะเมื่อเกิน free tier

Q: Free tier ของ Gemini พอไหม?

สำหรับการเล่นเกม 2-4 ชม./วัน เพียงพอ — ถ้าไม่พอสามารถอัปเกรดแผนได้
